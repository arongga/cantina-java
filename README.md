# Cantina Java

Menú de cantina desarrollado en Java con submenús anidados y sistema de pago.

## Funcionalidades
- Menú principal con 4 categorías
- Submenús de Refrescos, Almuerzos y Postres
- Total acumulable por pedido
- Sistema de pago y salida

## Tecnologías
- Java
- Eclipse IDE
